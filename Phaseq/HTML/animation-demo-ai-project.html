What is CSS Animation?

CSS Animation is a technique that allows you to create dynamic, engaging visual effects on web pages without using JavaScript. It enables HTML elements to transition from one style to another over a specified duration.

CSS animations can animate transitions from one CSS style configuration to another. Animations consist of two components: a style describing the CSS animation and a set of keyframes that indicate the start and end states of the animation.

ðŸ”‘ Key Concepts

1. @keyframes Rule

The @keyframes rule is the foundation of CSS animations. It defines the animation sequence by specifying styles at various points during the animation. You can define multiple keyframes using percentage values (0% to 100%).

@keyframes animationName {
    0% { 
        /* starting style */
    }
    50% {
        /* middle style */
    }
    100% {
        /* ending style */
    }
}

2. Animation Properties

Property Description Example Values
animation-name Specifies the name of the @keyframes fadeIn, moveRight, rotate
animation-duration How long the animation takes (seconds/s or milliseconds/ms) 2s, 500ms
animation-timing-function Speed curve of the animation ease, linear, ease-in, ease-out, ease-in-out, cubic-bezier()
animation-delay Delay before animation starts 1s, -1s (negative starts immediately)
animation-iteration-count How many times to repeat 1, 2, 3, infinite
animation-direction Direction of animation playback normal, reverse, alternate, alternate-reverse
animation-fill-mode Style when animation is not playing none, forwards, backwards, both
animation Shorthand for all properties name duration timing-function delay iteration-count direction

3. Animation Timing Functions

Timing functions define how the animation progresses through time:

  • ease - Starts slow, speeds up, then slows down (default)
  • linear - Constant speed throughout
  • ease-in - Starts slow, speeds up at the end
  • ease-out - Starts fast, slows down at the end
  • ease-in-out - Slow start, fast middle, slow end
  • cubic-bezier(n,n,n,n) - Custom bezier curve
  • steps(n) - Jump animation in n steps

Example 1: Fade In/Out Animation

This example demonstrates opacity animation - the box fades in and out repeatedly.

Fade Animation
.fade-box {
    animation: fadeIn 2s ease-in-out infinite;
}

@keyframes fadeIn {
    0% { opacity: 0.2; }
    50% { opacity: 1; }
    100% { opacity: 0.2; }
}

Example 2: Moving Animation (Translate)

This example shows horizontal movement using transform: translateX()

.move-box {
    animation: moveRight 3s ease-in-out infinite;
}

@keyframes moveRight {
    0% { transform: translateX(0); }
    50% { transform: translateX(300px); }
    100% { transform: translateX(0); }
}

Example 3: Rotating Animation

Using transform: rotate() to create continuous rotation

Rotate
.rotate-box {
    animation: rotate 4s linear infinite;
}

@keyframes rotate {
    0% { transform: rotate(0deg); }
    100% { transform: rotate(360deg); }
}

Example 4: Scaling Animation

Using transform: scale() to make elements grow and shrink

Scale
.scale-box {
    animation: scaleUp 2s ease-in-out infinite;
}

@keyframes scaleUp {
    0%, 100% { transform: scale(1); }
    50% { transform: scale(1.5); }
}

Example 5: Color Changing Animation

Animating background-color through multiple colors

ðŸ“‹ Summary

CSS Animations provide a powerful way to add motion and interactivity to your web pages. Key takeaways:

  • @keyframes defines the animation sequence with start, middle, and end states
  • animation properties control timing, duration, direction, and repetition
  • transform properties (translate, rotate, scale) create movement effects
  • transitions are simpler and work on hover/active states
  • Combine animations for complex effects
  • Use negative delay to start animation from middle
  • animation-fill-mode: forwards keeps the final state after animation ends

ðŸ’¡ Pro Tips

  • Use will-change: transform, opacity; for performance optimization
  • Prefer transforms over changing left/top/right/bottom for smoother animations
  • Use animation-direction: alternate for smooth back-and-forth motion
  • Combine with CSS variables for reusable animations
  • Use cubic-bezier() for custom timing functions
  • Consider reduced motion preferences for accessibility
